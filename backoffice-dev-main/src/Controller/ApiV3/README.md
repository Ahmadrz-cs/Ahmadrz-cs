# What is APIv3

- Intended as a modern PHP rewrite of a granular PHP that APIv1 originally implemented
- Deprecates APIv2 which offered a more abstracted experience
- APIv3 is currently a prototype and is not ready for production usage
- Many of APIv3's trappings can be found in newer rewrites to APIv1, with lessons from that to be brought forward to APIv3 (if that ever happens)
